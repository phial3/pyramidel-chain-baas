# 2023.3.9

密钥托管实现思路:
托管实现方式分为两步：
1.使用用户密码对私钥进行AES/SM1加密
2.使用加盐sm3散列用户密码(类似于sha256算法,理论上不可逆)，将盐值返回给用户保存，使用时用户上传盐值与密码，验证密码相同后进行私钥解密获取私钥内容
